# Pretty .bashrc
Custom .bashrc for linux and WSL

Two-line prompt:
   user@host [HH:MM] dir 
       └──────────────┴──$

Colors:
   user/host   = green (non-root) / red (root)
   time        = uncolored (fixed width and bracketed HH:MM)
   dir         = blue (writable) / red (non-writable)
   arrow       = dir color
   prompt char = user color
